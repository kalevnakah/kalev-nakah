Exchange Rate Calculator

Live Demo GitHub

Problem

An api opens the possibilities to extract data from third parties and build dynamic apps. Using https://open.exchangerate-api.com we can figure out what the exchange rates for doing international transactions.

Action

Use fetch with the API to get the exchange rate in a json file and build app that shows the rate based on countries you choose.

Result

User friendly, simple, and responsive exchange rates on command.

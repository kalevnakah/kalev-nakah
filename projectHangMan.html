Hang Man

Live Demo GitHub

Problem

Everyone's favorite past time games include the classics like tic-tac-toe and hangman. Hangman is great game to help you with your spelling and vocabulary. Build a dynamic app that recreates that classic game in your browser.

Action

In this game I built out the hangman app that uses html SVG tags to draw the hangman. And use the event listener to listen to key strokes and process the letters to determine if the match the randomly picked word from an array. If wrong letter is pressed it is added to an array of correct letters and the blanks are filled in. If the wrong letters are entered they get put in another array and displayed to the user. If a letter is repeated a notification pops up and lets the user know.

Result

Hours of fun educational learning from the comfort of your browser.

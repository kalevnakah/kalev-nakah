WordPress Sites

Problem

Many non-profit organizations and startups need a website. They need something that is going to be affordable and easy to quickly update by a novice user. It also should have the ability to scale as the organizations grow. It should also be flexible enough to take on new functions such as eCommerce or widgets.

Action

Word Press is an excellent solution as it can be deployed on almost any server for free, can easily configured for novice editors, scale and grow. With Elementor and Divi the possibilities are endless for creating professional looking websites.

Result

I created two WordPress sites using Elementor theme plugin for real world Non-Profits that I volunteer at. These sites connect staff, volunteers, students and parents with the correct information and the write people. I also created two WordPress sites with Divi plugin for two startup companies that I am working on.They are a simple landing page to test receptiveness and need for the products.

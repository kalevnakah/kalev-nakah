Menu Slider & Modal

Live Demo GitHub

Problem

Two very common widgits on a website is a menu that slides out and modal popup box. Adding these to site can add functionality without distracting from the main content.

Action

Using java script we can manipulate the css to change the display property to hide and unhide entire div's. Add some transitions and these boxes slide or pop into existence and disappear on command.

Result

A user interface that is familiar and displays information in an interactive way. The menu and form do not distract from the main contact, but are still accessible.

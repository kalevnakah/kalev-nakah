Problem

Forms are great but they can be frustrating when users or hackers enter the wrong information breaking the site or making the user frustrated.

Action

Using Javascript we can prevent the wrong data from being entered, display messages, and highlight inputs to instruct the user what needs to be entered.

Result

User friendly, simple, and responsive form that doesn't break the app or the user.

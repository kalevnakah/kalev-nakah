20 Web Projects With Vanilla JavaScript

Live Demo GitHub

Problem

In order to solidify my JavaScript knowledge I needed to work on some real projects and not just simple lessons dealing with only one element. Udemy offers many online classes. One of which is called "20 Web Projects With Vanilla JavaScript" by Brad Traversy who is know for his excellence in teaching people in how to code. I purchased his class and using it learn how to build apps that use the most common JavaScript elements and techniques.

Action

Use Visual Studio Code to write the projects in standard HTML, CSS, and JavaScript. Each project is broke down into three phases. 1. Build out the html elements. 2. Style elements. 3.Program the logic. The lessons are all video. In-order to prevent my self from just playing copycat, I changed all the names of the classes, id, variables and functions.

Result

Through this process I learned how all the different elements work together to create fully functional apps, a system for building apps from the ground up, and how to debug code.

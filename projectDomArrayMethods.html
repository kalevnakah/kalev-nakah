DOM Array Methods

Live Demo GitHub

Problem

Java array methods are extremely important and useful. The most common are map, sort, filter, and reduce. Need to make an app that utilizes all of them.

Action

Connect up to an api that randomly generates users. https://randomuser.me/ Randomly generate a number that represents their net worth. Use array methods to display the data.

Result

Used a fetch request to get a random user. Used "forEach" to cycle through the users and display them. Used "map" to double the net worth of the users. Used "sort" to organize the users from richest to poorest. Used "filter" to filter out and leave only the millionaires. Used "reduced" to calculate the sum total of all the users.
